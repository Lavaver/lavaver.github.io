如何为你的 iPhone 越狱

如果你在这里下载了需要越狱才能安装的 IPA ，那么你来对地方了。本文章教你如何为你的 iPhone 越狱。

⚠ 警告
如果你决定要为你的 iPhone 越狱，你需要具备基础的越狱常识，并关闭可能的设备锁防止变砖。同时，你手头还需要一台已经安装了 MacOS 的电脑。本文档将以 checkra1n 举例。

首先安装 checkra1n

  • 你需要拥有一台运行 MacOS 的计算机（不要求是 Mac 设备，但要求 MacOS 系统）

  • 从 checkra1n 中获取最新的 Beta 版。点击 Get the Beta now

  • 在进入的页面点击 checkra1n x.xx.x beta 字样的链接，进入后往下拉

  • 点击 Download for macOS ，系统将开始下载 .dmg 映像文件

  • 我们这里已经提供了一个官方下载链接。你可以点击此处进行下载

  • 下载完成后打开映像文件，将 checkra1n 图标移动到旁边的应用程序文件夹，开始安装

  • 安装完成后你就可以在启动台看到 checkra1n 了

开始越狱

  • 确保设备锁关闭。防止你的设备无法越狱或彻底变砖头

  • 链接 iPhone 到电脑

  • 点击 Options ，勾选 Allow untested iOS versions ，返回

Tips
如果你运行的 iOS 版本已测试，你无需进行此步骤。未经测试的 iOS 版本可能会在稳定性方面有所动摇

  • 点击 Start 按钮，开始越狱

  • 点击 Next 按钮，继续

  • 现在，checkra1n 应该已经开始将手机进入恢复模式了，请耐心等待

  • 当切换为下一个页面时，点击 Start 按钮，然后立即根据图片或本文档 Tips 提示进入 DFU 模式

对于 iPhone X / 8 及更新的型号
请快速按下并释放 音量增大 按钮，再快速按下并释放 音量减小 按钮。按住侧边按钮（电源按钮）直到屏幕变黑。屏幕变黑后，立即同时按住 侧边按钮 和 音量减小 按钮 5 秒钟。5 秒后，松开 侧边按钮，但继续按住 音量减小 按钮约 10 秒钟
对于 iPhone 7 / iPhone 7 Plus / iPhone 6s 及更早的型号
请同时按住 音量减小 和 电源 按钮 8 秒钟，8 秒后，松开 电源 按钮，但继续按住 音量减小 按钮约 10 秒钟

  • 如果你成功在规定时限内进入 DFU 模式，你会看到 checkra1n 正在跑进度条。这意味着正在进行越狱

  • 完成后自动返回第一页，手机将自动重启

  • 安装 Cydia ，更新必要的软件包

  • 返回本 IPA 下载站，下载需要越狱才能安装的应用，然后安装
