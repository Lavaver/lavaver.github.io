（必读）注意事项

  • 你手头有至少一个电子邮箱或者 QQ 账号。整个上传流程会涉及如下步骤。你可以参照这张流程图：

整个上传流程不需要你登陆这里的账号，这里也没有账号系统。所以在上传软件之前请仔细核对好你要上传的电子邮箱账户或 QQ 账户。因为该流程图一些关键流程需要以你上传软件时所用的电子邮箱地址或 QQ 账户进行回复关键进展。

  • 你不能上传危害设备安全的软件
  • 如有可能，请在上传时一并附上与软件强相关的图片

即使你未上传图片，我们也会尽可能的从 bing 中搜索与之软件强相关的图片。

  • 请不要上传 ≥3GiB 的软件，在电子邮件附件上传 ≥100MiB 的软件时需额外注意电子邮件提供商对于附件大小的限制

  • 在上传类似于证书类的软件时需谨慎
  • 在上传之前你或许可能需要检查下文件名是否带有不适宜内容（每次审核软件都会遇到过一两个文件名不太正常但内容正常的软件，这可能会对我们审核判断有一定影响）
  • 不要从不可信来源获取软件并上传上去
  • 如果该软件是一个开源软件，我们强烈欢迎你附上 GitHub 仓库链接，这有可能增加审核通过几率（因为作者愿意公开源代码就意味着我们至少有 80% 的信任可以直接通过审核）
  • 核对好这个软件对应的 iOS 版本并告诉我们，以便让我们部署时将其划分到合适的分区
  • 上传并审核通过后的软件一般今日就会开始部署。但要部署的文件过多时会分批进行部署，请谅解
