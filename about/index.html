网站时间加载过长或出错

因为 GitHub Pages 架设在国外，部分地区的访客可能因某些奇♂妙的原因导致加载时间过长甚至无法访问的问题（并且 CDN 的资金支持也是个问题喵…）

如果你正好碰见了这个原因，请用以下方法逐个排查：

  • 切换手机热点重试（经济且成功率又大，推荐使用，只是频繁用会烧流量）

  • 修改 Hosts 文件至可以连接的最快地址重试（需要有一定系统基础，涉及修改系统文件，操作前请备份）

  • 确认该地区是否可以裸连 GitHub（裸连：不通过加速器、修改 Hosts 等方式能直接访问 GitHub 的方式）（如果能裸连最好）

  • 使用加速器加速（看加速器本身情况，如果需要 git push 或其他类似需求咱还是换个方法哈）

  • 使用系统代理（除非万不得已不能使用，使用过后需要关掉代理，以免访问正常网页找不到域名服务器（DNS））
